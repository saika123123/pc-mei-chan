PC Mei - インストールマニュアル

はじめに

PC Meiとは，MMDAgentというソフトウェアとChrome Webブラウザを連携させ， あなたの日常生活に様々な支援を行うサービスです．
PC Meiという名前は"Personal Caregiver for Managing Elderly In-home-life"の略称で，「在宅高齢者の生活支援のための介護者」という意味です．

Chromeだけでも動作可能ですが，MMDAgentを連携させる場合には，お手元のPCに下記の実行環境を構築する必要があります．

  • Windows10のPC
  • Java 実行環境
  • Apache Tomcat Webサーバ
  • MMDAgent ToolKit (神戸大学版)

Java実行環境のインストール

  1. こちらを参考に，お手元のWindows PCにJava環境 (JDK14)をインストールしてください．
  2. 上記リンク内の説明にあるように，環境変数JAVA_HOMEの設定と，PathへのJavaの追加も忘れずに行ってください

Apache Tomcatのインストール

  1. こちらからzipファイルをダウンロードしてください．
  2. zipファイルを解凍して，展開されたtomcatフォルダをCドライブ直下にコピーしてください
    • C:\tomcatの下にbin, conf, lib, ... となるように配備する
  3. エクスプローラで，C:\tomcat\binを開き，startup.batをダブルクリックで実行してください．
  4. 新しくウィンドウが開き，実行ログが表示されます．最後に，サーバーの起動 [nnnn]msと出ればOKです．

MMDAgentのインストール

  1. こちらからzipファイルをダウンロードしてください．
  2. zipファイルを解凍して，展開されたMMDAgentフォルダをデスクトップに移動してください
  3. エクスプローラで，MMDAgentフォルダの中のMMDAgent_Meiフォルダを開きます
  4. MMDAgent_Example.mdfを右クリック→プログラムから開く
  5. 「このファイルを開く方法を選んでください」→「その他のアプリ」→最下部までスクロール→「このPCから探す」
  6. デスクトップ\MMDAgent\MMDAgent_win32-1.4\MMDAgent.exeを選択して，開く
  7. メイちゃんが起動したら成功です．

テストする

  1. こちらをクリックしてメイちゃんが手を振るか確認します
  2. こちらをクリックしてメイちゃんが「こんにちは」としゃべるか確認します
